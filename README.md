# PersonalLedger
开发一个命令行交互的个人记账工具，支持收入/支出记录的增删、分类统计、收支结余计算，并将数据持久化到本地文本文件。用于巩固C++核心语法、类设计、文件I/O及调试能力。
### 基于C++的个人记账本

`C++17`  `STL容器(vector)`  `面向对象设计` 



- **项目描述**：
  开发一个命令行交互的个人记账工具，支持收入/支出记录的增删、分类统计、收支结余计算，并将数据持久化到本地文本文件。用于巩固C++核心语法、类设计、文件I/O及调试能力。

- **工作内容**：
  
  - 设计 Record类封装单条记录（日期、类别、金额、备注），Ledger类管理 vector<Record> 器及业务逻辑。
  - 实现添加记录、查看所有记录（按时间倒序）、按类别筛选、统计总收入/支出/结余等功能。
  - 自定义文件存储格式（日期|类别|金额|备注），使用 std::fstream 现数据加载与自动保存，确保程序重启后数据完整恢复。
  - 采用模块化菜单循环，对用户输入（金额格式、空记录等）进行基础校验，避免程序崩溃。

- **项目难点**：
  
  - 文件解析与序列化：处理含特殊字符（如备注中的空格、竖线）的读写，统一编码规范。
  - 容器与迭代器的使用：删除记录时保证迭代器有效，避免 vector 失效问题。
  - 统计逻辑：正确区分收入/支出（金额正/负），并支持多类别聚合计算。

- **个人收获**：
  
  - 深入理解了C++类封装、STL容器和文件流的实际应用。
  - 学会设计简单的文本存储格式，为后续迁移到 SQLite 或使用 Qt 做图形界面打下基础。
  - 提升了代码调试能力（通过输出中间状态、断言检查），能够独立完成一个小型完整项目。
